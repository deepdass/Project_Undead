# Gaspfornow (WIP)
This is a project that is made by modifying the Game animation sample project 5.5 provided by Epic games for my use case
## Idea
A fast paced parkour Zombie Game, with some rpg aspect and bunch of customization that you could do
<br>I have many ideas and mechanics planned for this project, but I will add them here as I work on them. (some of them are my own and some are inspired by the games i love )

## Why 
I love Zombie and fast paced game. also games which give many options on how they wanna play the game, such as weapon customization and playstyle choices.

## Progress
I come back to this project as I learn more about the game development pipeline and learn new skills that could be useful to this project like after learning asset creation or animation etc
